What is Notepad++?

Notepad++ is a free source code editor (not an IDE), runs on MS Windows environment and supports several languages.

Read more about it at: notepad-plus-plus.org.

Is Notepad++ good?

It has many built in tools:

  • Marco.
  • Multi-editing.
  • Encode/Decode Base64.
  • ...

So it will depend on your purpose.

An Easter Egg!

  1. Open Notepad++ and create new file.

  2. Write "Space Invaders" and select all of the words.
